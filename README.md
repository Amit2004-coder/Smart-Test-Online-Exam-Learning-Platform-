# Smart-Test-Online-Exam-Learning-Platform-

**Smart Test Website** is an online learning and practice platform designed to help students test their knowledge, improve their skills, and track their learning progress.

This project was developed using **HTML, CSS, JavaScript, PHP, and MySQL** to create a simple and interactive environment where learners can practice questions, watch lectures, and receive instant feedback on their performance.

The platform provides structured practice through different difficulty levels and a large collection of questions across multiple subjects.

## ✨ Features

* 📚 **12,000+ Practice Questions** across multiple subjects
* 🎯 **Three Difficulty Levels**

  * Beginner
  * Intermediate
  * Advanced
* 🎥 **Video Lectures** to help understand concepts better
* 📝 **Study Notes** for quick revision
* 🧪 **Online Tests** with instant results
* 📊 **Performance Feedback** after completing tests
* 🏆 **Certificate Generation** after successful test completion
